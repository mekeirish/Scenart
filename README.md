# 🎭 Scenart

> Une Progressive Web App (PWA) de pointe conçue pour réinventer l'expérience utilisateur à travers une interface immersive, fluide et hautement interactive.

Développée selon le protocole **PWA-CORE**, **Scenart** rassemble toute sa logique, ses styles et sa structure au sein d'une architecture monolithique optimisée. Conçue spécialement pour le mobile-first et le "vibe coding", l'application offre des performances fulgurantes et une maintenance ultra-simplifiée par IA.

---

## ✨ Fonctionnalités Clés

- 📱 **PWA Native & Mobile-First** : Installable instantanément sur iOS, Android et Desktop avec support complet du mode hors-ligne.
- 🎨 **Interface Premium** : Design skeuomorphique/neumorphique immersif avec des textures réalistes, des effets 3D et une gestion native du mode sombre.
- ⚡ **Performance Monolithique** : Tout le code (HTML5, CSS3, JavaScript) est centralisé dans un fichier unique pour un chargement immédiat et une réactivité maximale.
- 🔄 **Synchronisation & Persistance** : Gestion robuste des données locales pour garantir une fluidité d'utilisation en toutes circonstances.

## 🛠️ Stack Technique

- **Frontend** : HTML5, CSS3 (Variables dynamiques, Flexbox/Grid, effets 3D)
- **Logic & Core** : JavaScript ES6+ (Architecture réactive, Vanilla JS)
- **PWA** : Service Workers, Web App Manifest, Stockage Local
- **Philosophie de dév** : Vibe Coding & PWA-CORE Architecture

## 🚀 Installation & Utilisation Rapide

Comme l'application utilise une architecture monolithique propre, aucun outil de build complexe (`npm install`, `webpack`, etc.) n'est requis.

1. **Récupérer le projet** :
Télécharge ou clone directement ce dépôt GitHub sur ta machine locale.

2. **Lancer l'application** :
Ouvre simplement le fichier `index.html` dans ton navigateur, ou utilise une extension de serveur local (comme *Live Server* sur VS Code).

3. **Installer en tant que PWA** :
Clique sur l'icône d'installation dans la barre d'adresse de ton navigateur (Chrome/Edge) ou utilise l'option "Sur l'écran d'accueil" (Safari sur iOS).

## 📐 Architecture PWA-CORE

Ce projet est structuré selon un modèle de fichier unique (`index.html`). Cette approche garantit :
- Une **lisibilité maximale** par les agents et assistants IA pour des itérations et modifications ultra-rapides.
- Une **absence totale de friction au déploiement** (Zero-configuration).
- Une **cohésion parfaite** entre le design UI et la logique applicative.
